5. Complete Commands (Step-by-Step)
Step 1 – Clone Repository
git clone repository_url
cd repository_name
Step 2 – Create Child Branch
git branch childbranch
git checkout childbranch

OR

git checkout -b childbranch
Step 3 – Work in Child Branch
git add .
git commit -m "feature added"
git push origin childbranch
Step 4 – Merge Child Branch to Main
git checkout main
git pull origin main
git merge childbranch
git push origin main
6. Full Workflow Example
git clone https://github.com/user/project.git

cd project

git checkout -b childbranch

git add .
git commit -m "new feature"

git push origin childbranch

git checkout main

git merge childbranch

git push origin main

✅ Important DevOps Concept

Branch	Purpose
main	Production code
childbranch	Feature development
merge	Combine feature into main
